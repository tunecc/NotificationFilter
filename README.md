`Notification Filter` 是一个面向越狱设备的通知过滤插件。  

适合这类场景：

- 禁用某些TG群组Pin消息的通知
- 含有某些关键字的通知
- 一些软件无法关闭的推广通知

---

## 特性

### 1. 全局规则 + 单应用规则

- 匹配顺序为：

  **全局规则**    --> **单应用规则**

  命中**排除规则**时会直接放行

### 2. 规则类型

- **包含规则**
- **排除规则**
- **正则规则**

### 3. 匹配范围

每条规则都可以指定匹配范围：

- 正文 / `message`
- 标题 / `title`
- 副标题 / `subtitle`
- 全部文本

### 4. 通知中心清理

可选开启“从通知中心删除过滤通知”：

- 开启后，已过滤通知会从通知中心移除
- 未开启时，禁用插件，注销后，历史过滤通知会在通知中心恢复显示

### 5. 已过滤通知日志

内置“已过滤通知”页面，支持：

- 搜索应用、规则和通知内容
- 查看命中的规则、时间、应用和通知详情

### 6. 规则导入导出

支持将当前配置导出为 JSON，也支持从 JSON 导入：

- 导出到文件
- 复制配置 JSON
- 从剪贴板导入
- 手动粘贴 JSON 导入

适合备份、迁移和分享规则。
